Chef Remy

Chef Remy is a custom recipe generating fullstack website and Progressive Web App (PWA) with a companion Alexa Skill. Visitors can navigate through web pages using Alexa or with the browser based Speech Recognition button on the home screen. Users can upload a photo of their store receipts to receive a list of the items they bought, which can immediately be added to their fridge and used to find their personal recipe suggestions.

Technologies: Javascript, React, Redux, Node, Express, Web Speech API, Edamam API, Spoonacular API, Sequelize, Tesseract, PSQL, AWS Lambda

Arrest Alert

A fullstack website for alerting friends and family when you've been arrested. Users fill out their emergency contacts' info along with a message, which automatically gets sent out via the Twilio API if their location is detected, via Google Place API, as being in police custody.

Technologies:

React, Redux, Node, Express, Sequelize, PSQL, Twilio API, Google maps api, Google place API
